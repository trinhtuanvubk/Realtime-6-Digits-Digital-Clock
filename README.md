# Realtime-6-Digits-Digital-Clock
- 6 Digits 7 Segment LED Digital Clock with RTC
- Use Asembly 
- Format : Hours/Minutes/Seconds 
- Components : MCS51 microcontroller, RTC DS1307, 7-segment display, logic buttons
